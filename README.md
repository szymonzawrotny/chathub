# 💬 Chat Application

A real-time chat application inspired by Discord and Slack, supporting channels and private messaging. Built with a focus on clean UI, responsiveness, and scalable architecture.

---

## ✨ Features

* ⚡ Real-time messaging
* 👥 Private and group conversations
* 📁 Channel-based communication
* 🔔 Notifications system
* 📱 Responsive modern UI
* 🔐 Authentication system
* 🌙 Clean and scalable architecture

---

## 🛠️ Tech Stack

### Frontend

* Next.js
* TypeScript
* Scss

### Backend

* Node.js
* NestJS
* WebSockets / Socket.IO

### Database

* PostgreSQL

---

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/szymonzawrotny/chathub
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```
